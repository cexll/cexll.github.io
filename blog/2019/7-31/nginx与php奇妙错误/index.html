Nginx与PHP奇妙错误

Posted Jul 31, 2019

Nginx 与 PHP 的配置

location ~ \.php$ {  
root html;  
fastcgi_pass 127.0.0.1:9000;  
fastcgi_index index.php;  
fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;  
include fastcgi_params;  
}

出现 File not found.

进入nginx.conf配置文件修改

astcgi_param SCRIPT_FILENAME /usr/share/nginx/html$fastcgi_script_name;

这里的路径为自己的Web环境路径,修改后重启Nginx服务

出现 No input file specified

  1. 修改php.ini里配置

    php --ini #获取路径

找到cgi.force_redirect

cgi.force_redirect = 1

修改后重启php-fpm服务
