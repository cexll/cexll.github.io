go mod

GOMODEL绝对是一个非常棒的,这里使用小案例讲解如何使用go mod

mkdir hello
cd hello
touch main.go
# 初始化go mod
go mod init go.mod
vim main.go

  package main
  import "github.com/gin-gonic/gin"
  func main() {
  	  r := gin.Default()
	  r.GET("/ping", func(c *gin.Context) {
		  c.JSON(200, gin.H{
			  "message": "pong",
		  })
	  })
	  r.Run() // listen and serve on 0.0.0.0:8080
  }

go run main.go
这里就会自动去下载需要的库,并且在`go.mod` require 包的版本路径,生成`go.sum` 生成依赖库的信息
